# Utility Scripts

Data mappings and version helpers. (Dependency updates moved to the ops CLI —
run `pnpm ops dev:update-deps`.)

## Files

- **bump-version.sh** - Bump the monorepo version (via `pnpm bump-version`)
- **uuid-mappings.json** - UUID mappings for v2 to v3 migration (not tracked)
- **uuid-mappings.example.json** - Example UUID mappings structure
