# Archive

Historical and obsolete scripts kept for reference.

## Subdirectories

- **qdrant/** - Scripts for the old Qdrant vector database (v2 used Qdrant, v3 uses pgvector)
- **v2-migration/** - One-time migration scripts from v2 to v3 (Shapes.inc data import, etc.)

**⚠️ These scripts are not maintained and may not work with current v3 architecture.**
